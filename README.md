# TestAbd

**TestAbd** is a social quiz and learning platform built with Flutter. Users can answer and create quiz questions, read books, follow other learners, and compete on leaderboards — all in one app.

## Features

- **Quiz Feed** — Scroll through questions shared by people you follow. Answer single-choice, multiple-choice, and true/false questions with instant feedback.
- **Test Mode** — Swipe-based focused test session for rapid-fire practice.
- **Create Content** — Build your own question blocks and share them with the community.
- **Library** — Browse and read books (PDF & EPUB). Chat with an AI assistant after finishing a reading session.
- **Social** — Follow users, view profiles, and see followers/following lists.
- **Leaderboard** — Compete with other learners and track your ranking.
- **Bookmarks** — Save questions to revisit later.
- **Notifications** — Real-time push notifications via WebSocket.
- **Multilingual** — Supports English, Russian, and Uzbek.
- **Dark / Light theme** — Full theme support.

## Tech Stack

- **Flutter** (Dart)
- **BLoC / Cubit** — state management
- **GetIt + Injectable** — dependency injection
- **Dio** — networking
- **Hive** — local storage
- **go_router** — navigation
- **pdfrx / flutter_epub_viewer** — book reading
- **WebSocket** — real-time notifications
